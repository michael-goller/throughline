// get_started

Three commands. One deck.

Install Shine, pick your favourite AI agent, describe your deck, and present.

1

Install the CLI

One-liner for macOS, Linux, or WSL. Needs Node 20+.

2

Pick your AI agent

Works with Claude Code, Gemini CLI, OpenAI Codex, or any coding agent.

3

Create and present

Describe what you want in natural language. Open in your browser.

You don't write slides. You describe a talk. Tools like Marp and Slidev let you write slides in markdown. Shine is different — you describe what you want in plain language, and your AI agent builds the deck. Layout, structure, styling, transitions. You handle the story.
// under_the_hood

Shine is just a web app built for decks.

No proprietary format. No lock-in. Your slides are a TypeScript config rendered by React with 30+ built-in slide types.

30+ Built-in Layouts

Title, metrics, timelines, Gantt charts, comparisons, scorecards, and more — ready to use out of the box.

Works with Any AI Agent

Claude Code, Gemini CLI, OpenAI Codex — any agent that can edit a TypeScript file can build your deck.

PDF & PNG Export

One command to export your entire deck as a PDF or individual slides as PNGs. Share anywhere.

Speaker notes & timer Keyboard navigation Live reload Drag-and-drop editor Dark/light themes Smooth transitions Slide overview
