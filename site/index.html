// under_the_hood

Shine is just a web app built for decks.

No proprietary format. No lock-in. Your slides are a TypeScript config rendered by React with 30+ built-in slide types.

Plain TypeScript

Your deck is a slides.config.ts file — an array of slide objects. AI agents read and write it directly.

Modern Rendering

React 19, Tailwind CSS, and smooth animations. Dark & light themes, responsive on any screen.

30+ Built-in Layouts

Title, metrics, timelines, Gantt charts, comparisons, scorecards, and more — ready to use.

Navigate slides without leaving the keyboard Presenter mode PDF/PNG export See changes instantly as you edit Slide overview Visual editor Adapts to your system theme Feedback & reactions Deck gallery Polished transitions between slides Write slides in Markdown with React components 30+ built-in layouts for common slide patterns
