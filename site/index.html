// get_started

Three commands. One deck.

Install Shine, pick your favourite AI agent, describe your deck, and present.

1

Install the CLI

One-liner that sets up Shine on macOS, Linux, or WSL. Needs Node 20+.

$ curl -fsSL https://shine.sh/install | bash
# or via Homebrew:
$ brew install michael-goller/tap/shine
2

Pick your AI agent

Shine works with any coding agent that can read and write files. Set up the one you prefer.

Claude Code

Anthropic's coding agent. Deep reasoning, great for complex decks.

Setup guide →

Gemini CLI

Google's agent with large context window. Fast iteration.

Setup guide →

OpenAI Codex

OpenAI's terminal agent. Lightweight and effective.

Setup guide →
3

Create and present

Scaffold a new deck, describe what you want in natural language, and open it in your browser.

$ shine new my-quarterly-update
$ cd my-quarterly-update

# Tell your agent what to build:
$ claude "Create a Q1 review deck with revenue metrics,
  team highlights, and next quarter roadmap"

# Preview it live:
$ shine dev
# → Local: http://localhost:5173
// under_the_hood

Shine is just a web app built for decks.

No proprietary format. No lock-in. Your slides are a TypeScript config rendered by React with 30+ built-in slide types.

Plain TypeScript

Your deck is a slides.config.ts file — an array of slide objects. AI agents read and write it directly.

Modern Rendering

React 19, Framer Motion, and Tailwind CSS. Smooth animations, dark & light themes, responsive on any screen.

30+ Slide Types

Title, metrics, timelines, Gantt charts, comparisons, scorecards, and more — all built in and ready to use.
